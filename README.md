# type-adventure-game
This is the Code for the Type Adventrue Game 

<h1>Storyline</h1>
<p>You are a Respectable Knight in service of King Ubers</p>
<p>The Sword of Ubers passed on from multiple generations was stolen by a Dark Mage</p>
<p>You are tasked with retreiving the Sword of Ubers back from the Mage's castle</p>
<p>Good Luck!</p>
